# Build Your AI Agent  
**ACM Workshop – Resources Repository**

Welcome to the official resource repository for the **“Build Your AI Agent”** workshop conducted by **ACM**.

This repository contains all the materials shared during the session to help you understand, build, and extend your own AI agents.

---

## What You’ll Find Here
- Workshop slides and notes  
- Introduction to AI Agents and LLMs  
- Code examples (agents, tools, prompts)  
- Useful links and references  
- Optional experiments and extensions  

---

## Topics Covered
- What is an AI Agent?  
- Agent architecture (LLM + Tools + Memory)  
- Prompting and reasoning basics  
- Simple agent implementation  
- Real-world use cases
